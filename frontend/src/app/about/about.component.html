About the website


This website aims to provide relevant information on museum artifacts and artists of the Smithsonian American Art Museum (SAAM) based onsemantic querying. This application queries and fetches information about an artwork or an artist by search queries with keywords


To use of the website, follow these steps:
1. Enter your query in the search box
2. Select one option from the dropdown menu
3. Click submit to see the results
